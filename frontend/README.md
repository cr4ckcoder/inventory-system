
# Inventory System Frontend (Next.js 14 + Tailwind v4 + ShadCN UI)

This frontend is Docker-ready and built using Next.js 14 (App Router), TailwindCSS v4, and ShadCN UI.

## Development
```bash
npm install
npm run dev
```

## Docker
```bash
docker build -t cis-frontend .
docker run -p 3000:3000 cis-frontend
```
